Led the development (as part of Team 24) of a comprehensive software application for managing and simulating the Neureset medical device. Utilized C++ and Qt to implement features including real-time EEG graph visualization, session logging, and battery status monitoring.

Engineered the core functionality for treatment sessions, ensuring accurate data representation through advanced signal processing techniques.

Designed and implemented a user-friendly interface with intuitive navigation, enhancing user experience and interaction with the application.
