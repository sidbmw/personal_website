Contributed to the development of an application that detects and recognizes traffic signs in images using computer vision and deep learning techniques. (Based on the repository at [bardia-p/Traffic-Sign-Detection](https://github.com/bardia-p/Traffic-Sign-Detection)).

The system combines various methods including:
*   A Convolutional Neural Network (CNN) fine-tuned on the German Traffic Sign Recognition Benchmark (GTSRB) dataset.
*   Template Matching.
*   Scale-Invariant Feature Transform (SIFT) for enhanced accuracy.

Also contributed to creating a user-friendly GUI for image processing and result visualization.
